Final Project OOP Java — Sistem manajemen penumpang Bus Trans Koetaradja. Mengimplementasikan class, objek, encapsulation, polymorphism, collection, dan exception handling. Program mencakup fitur naik/turun penumpang, kursi prioritas, saldo kartu

# Bus Trans Koetaradja — OOP Java

Projek akhir Pemrograman Berorientasi Objek. Aplikasi ini mensimulasikan sistem bus Trans Koetaradja dengan fitur:
- Kursi prioritas (lansia, anak-anak, ibu hamil)
- Kursi biasa
- Penumpang berdiri
- Pembayaran saldo otomatis
- Simulasi penumpang otomat
